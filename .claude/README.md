# Thư mục .claude/

File ngữ cảnh chính cho Claude Code là **`CLAUDE.md` ở thư mục gốc** (không
phải trong thư mục này) — Claude Code tự động đọc file đó khi mở project.

Thư mục này để dành cho cấu hình cục bộ của Claude Code sau này nếu cần (VD
`settings.json` cho quyền hạn công cụ, hooks...) khi bắt đầu dùng Claude Code
thật sự trên máy. Hiện tại chưa cần — chỉ tạo sẵn thư mục để có chỗ khi cần
đến, tránh phải nhớ tạo lại.
